## 🚀 Comment déployer

### Option 1: GitHub Pages
1. Allez dans les **Settings** du repo
2. Descendez à **GitHub Pages**
3. Sélectionnez `main` branch comme source
4. Votre site sera live à : `https://irislorline-boop.github.io/esther-hounsinou-portfolio`

### Option 2: Vercel
1. Connectez le repo à Vercel
2. Déploiement automatique à chaque push

### Option 3: Netlify
1. Drag & drop le dossier sur Netlify
2. Ou connectez votre repo GitHub

## 🎨 Personnalisation

### Modifier les couleurs
Ouvrez `css/style.css` et modifiez les variables CSS au début :

```css
:root {
    --primary-color: #2c3e50;
    --secondary-color: #e74c3c;
    --accent-color: #3498db;
    /* ... autres couleurs */
}
```

### Modifier le contenu
Éditez `index.html` pour :
- Ajouter/modifier sections
- Changer textes et descriptions
- Ajouter vos projets réels
- Mettre à jour témoignages

### Email de contact
Pour faire fonctionner le formulaire, utilisez un service comme:
- **Formspree** - https://formspree.io
- **Netlify Forms**
- **EmailJS**
